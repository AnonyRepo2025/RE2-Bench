The `check_in` method returns `False` if the room type is not in `booked_rooms` or if the given name is not in the specified room type dictionary. It returns `None` if the room number is equal to the number of rooms booked under the given name.

To get `None` as output, we need to satisfy the condition `room_number == self.booked_rooms[room_type][name]`.
