The `update_product_quantity` method updates the quantity of a product in the inventory. The output is `null`, which means the function doesn't return anything.  The function modifies the `self.inventory` dictionary in place.

Let's consider a scenario:

Initial inventory:
```
self.inventory = {
    "1": {
        "name": "Laptop",
        "quantity": 10
    }
}
```

If we call `update_product_quantity("1", 5)`, the quantity of product "1" will be updated to 15 (10 + 5).

Since the output is `null`, we can choose any valid input that doesn't raise an error. Let's use the above example.
