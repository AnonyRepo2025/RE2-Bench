Let the input be N, A, B.
The output is calculated as follows:
ans = N // (A + B) * A
rem = N % (A + B)
ans += min(rem, A)

The output is 0.

Case 1: N = 0
ans = 0 // (A + B) * A = 0
rem = 0 % (A + B) = 0
ans += min(0, A) = 0

Case 2: A = 0
ans = N // (0 + B) * 0 = 0
rem = N % (0 + B)
ans += min(rem, 0) = 0

Let's choose N = 0, A = 1, B = 1.
