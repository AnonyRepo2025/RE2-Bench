The `insert_into_database` function inserts data into a specified table in the SQLite database. The output is `null`, meaning the function doesn't return any value, only performs the insertion.  We need to provide the `table_name` and the `data` to be inserted. The `data` should be a list of dictionaries, where each dictionary represents a row with 'name' and 'age' keys. Let's assume we insert two rows into a table named 'people'.

[/THOUGHT]

[INPUT]
```json
{
    "self": {
        "database_name": "test.db"
    },
    "args": {
        "table_name": "people",
        "data": [
            {"name": "John Doe", "age": 30},
            {"name": "Jane Doe", "age": 25}
        ]
    },
    "kwargs": {}
}
```
